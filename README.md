# Polling System

A real-time polling application for classrooms.

## Features
- **Teacher Dashboard**: Create polls, view active results, manage queue, chat, and participants.
- **Student Portal**: Join sessions, vote in real-time, view live results.
- **Resilience**: Handles disconnects and refreshes state properly.

## Tech Stack
- Frontend: React, Vite, TailwindCSS
- Backend: Node.js, Express, Socket.IO
- Database: MongoDB
